Sparks App

* The purpose of this app is to provide users with an easy to use app that assists users in suggesting social connections/interactions during busy times based on the time of day.

* The design is simple, easy to use, and fun to use.

* The app features
  Input of time of day
  Get personalised social sparks
  Reset button clearing the input time as well as the suggested spark
  Error messages as well as positive feedback based on incorrect inputs

  *The use of Github and Github actions
  The ability to switch to different versions
  Record of code changes overtime
  Applicication is complete
  Tests are completed
